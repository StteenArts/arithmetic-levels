# 🧠 Centro de Entrenamiento Técnico (Console App)

Este proyecto es una aplicación de consola desarrollada en C# que simula un sistema de entrenamiento por niveles. Cada nivel representa un conjunto de ejercicios enfocados 
en mejorar habilidades de programación, lógica y manejo de datos.

---

## 🚀 Características

- Menú interactivo por consola
- Validación de entradas del usuario
- Estructura modular
- Progresión por niveles de dificultad
- Uso de clases y métodos externos

---

## 🧩 Niveles del sistema

El programa está diseñado para escalar en complejidad:

1. **Suma y validación numérica**
2. Cálculo y condicionales *(en desarrollo)*
3. Manipulación de cadenas *(en desarrollo)*
4. Operaciones lógicas (calculadora) *(en desarrollo)*
5. Análisis numérico con listas
6. Gestión de tareas (listas de strings)
7. Arreglos y búsqueda
8. Programación orientada a objetos
9. CRUD de objetos
10. Salida del sistema

---

## 🛠️ Tecnologías utilizadas

- C#
- .NET (Aplicación de consola)
- Programación orientada a objetos

---

## 📂 Estructura del proyecto

- `Program.cs` → Control principal del flujo del programa
- `Modules/arithmetic.cs` → Lógica de operaciones matemáticas
